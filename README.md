# 微信消息推送

实现每日向固定的人推送一条暖心消息

## 环境准备

* 操作系统：支持 macOS，Linux，Windows

* 运行环境：建议选择 [LTS](https://nodejs.org/en/) 版本，最低要求 8.x。


## 快速开始

安装依赖

```
npm i
```

启动项目

```
npm run dev
open http://localhost:7001
```

单元测试

```
npm test
```
